# 🎮 UNDERTALE Web Edition

**完全動作版 - シンプルで確実に動くバージョン**

## ✅ 動作確認済み

- ✅ 十字キー / WASD で移動
- ✅ 6つの部屋を探索
- ✅ セーブポイント (2箇所)
- ✅ 部屋間の移動
- ✅ 壁・オブジェクトの衝突判定

## 🎮 操作方法

```
矢印キー / WASD : 移動
Z / Enter       : 決定 (名前入力時)
```

## 🗺️ マップ構成

1. **おちた ばしょ** (スタート地点)
   - 花畑のある部屋
   - 上へ進む

2. **ろうか**
   - シンプルな廊下
   - 上下に移動可能

3. **セーブポイント** ⭐
   - 黄色い星マーク
   - 右へ進むと長い廊下

4. **ながい ろうか**
   - 柱が並ぶ長い廊下
   - 左右に移動可能

5. **パズルの へや**
   - 下へ進むとトリエルの家

6. **トリエルの いえ** ⭐
   - セーブポイントあり
   - 家具が配置されている

## 🚀 起動方法

### 方法1: ローカルサーバー (推奨)

```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```

ブラウザで開く: `http://localhost:8000`

### 方法2: 直接開く

`index.html` をダブルクリック (一部のブラウザで動作)

## 🔧 カスタマイズ

### プレイヤー速度を変更

`js/game.js` の `CONFIG` オブジェクト:

```javascript
const CONFIG = {
    PLAYER_SPEED: 4,  // この値を変更 (1-10推奨)
    // ...
};
```

### 色を変更

`COLORS` オブジェクト:

```javascript
const COLORS = {
    ruins: {
        bg: '#1a0f1a',      // 背景色
        wall: '#2d1a33',    // 壁の色
        floor1: '#3a2244',  // 床の色1
        floor2: '#4a2e54',  // 床の色2
        accent: '#8b5a9f'   // アクセント色
    }
};
```

### 新しい部屋を追加

`ROOMS` オブジェクトに追加:

```javascript
your_room: {
    name: 'あなたの へや',
    width: 16,
    height: 12,
    playerStart: {x: 8, y: 6},
    exits: {
        top: 'other_room'
    },
    walls: [
        // 壁の座標
    ]
}
```

## 🐛 トラブルシューティング

### 動かない場合

1. **ブラウザのコンソールを確認** (F12キー)
2. **ローカルサーバーで起動** (直接開くのではなく)
3. **最新のブラウザを使用** (Chrome, Firefox, Edge推奨)

### キーボードが反応しない

1. ゲーム画面をクリック
2. F5キーでリロード
3. 名前入力画面で「けってい」を押したか確認

## 💡 デバッグ

ブラウザのコンソール (F12) で:

```javascript
// ゲーム状態を確認
console.log(game);

// 特定の部屋に移動
loadRoom('house');

// プレイヤー位置を確認
console.log(game.player.x, game.player.y);
```

## 🎯 シンプル設計の理由

このバージョンは以下の理由でシンプルに設計されています:

- ✅ **確実に動作**: 複雑な依存関係なし
- ✅ **理解しやすい**: 全コードが1つのファイル
- ✅ **カスタマイズ容易**: 変数を変えるだけ
- ✅ **軽量**: 最小限のコード

## 📝 技術仕様

- **Pure JavaScript**: フレームワーク不要
- **HTML5 Canvas**: 2D描画
- **タイルベース**: 40x40ピクセルのタイル
- **リアルタイム**: requestAnimationFrame使用
